RBX Bot

RBXBot is a Mod and Fun Bot. It Has; A Backup System, Ecomany System, And Security System.

Commands

Economy:

Bal, Leaderboard/LB, Work | Server Admin Only: addmoney, setmoney, removemoney

Fun:

Say, 8Ball, Ascii, Meme, WB, RPS, Embed

Others:

Invite, Server-info, Info, Avatar, Ping, Support, Status

Admin:

Give-Role, Delemoji, Addemoji, Mute, Unmute, Ban, Kick

Backup System

RBXBot has a backup system where you can backup your Discord server channels. To back up channels do -backup and you get sent a DM with a ID with the info Of backups do -infos (ID HERE) And load backup with -load (ID Here)
